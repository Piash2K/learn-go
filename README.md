# 🚀 Learn Go

A simple repository for learning and practicing the Go programming language.

## 📚 About This Project

This repository contains my Go learning journey, including:

- Variables & Data Types
- Functions
- Conditions
- Loops
- Scope
- Closures
- Basic Problem Solving
- Go Syntax Practice

The goal of this project is to improve my understanding of Go fundamentals and backend development concepts.

## ▶️ Run Locally

Clone the project:

```bash
git clone https://github.com/Piash2K/learn-go.git
```

Go to the project directory:

```bash
cd learn-go
```

Run the project:

```bash
go run main.go
```
